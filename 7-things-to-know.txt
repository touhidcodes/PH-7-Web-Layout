For responsive 7 rule should be maintained:

1. Viewport Meta Tag (Default)
2. CSS Relative Unit (em , rem , % , vw)
3. Responsive Image (Fluid Image, width: 100%)
4. Body max-width and margin: auto
5. Responsive Split Screen Using Flexbox (Duo line)
6. Grid Responsive (Multi line)
7. Responsive Menu (Bootstrap, Javascript)
