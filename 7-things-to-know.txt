7 Things to Make Website Responsive

1. Use Viewport Meta Tag (Default)
2.  CSS Relative Measuring Unit (`em, rem, %, vw`)
3. Make Image Fluid (use `width: 100%`)
4. Body `max-width` and `margin: auto`
5. Two Column Using Flex Layout and Media Query
6. Multi Column Using Grid Layout and Media Query
7. Responsive Menu (Bootstrap, Java Script)
